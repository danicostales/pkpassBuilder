# Assets

Activos necesarios para la generación de pases.

## Estructura

- **cert/**: Certificados de Apple (⚠️ ignorados en .gitignore - no subir a git)
- **img/**: Imágenes, logos y diseños para los pases

## Certificados (SEGURIDAD)

Los certificados requeridos para firmar los pases NO deben subirse al repositorio. El `.gitignore` los ignora automáticamente:

```
*.p12
*.cer
assets/cert/
```

Para obtener los certificados, consulta el README principal.
