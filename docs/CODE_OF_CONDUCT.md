# Código de Conducta

Este proyecto adopta el Código de Conducta del Contributor Covenant v2.0 para fomentar una comunidad abierta, respetuosa e inclusiva. Esperamos que todos los participantes sigan estas normas.

Comportamiento esperado:

- Trata a los demás con respeto y consideración.
- Acepta críticas constructivas de forma profesional.
- Evita comentarios ofensivos, sexistas, racistas, homofóbicos o discriminatorios.

Cómo reportar un abuso:

Si experimentas o presencias un comportamiento inaceptable, por favor contacta con los mantenedores en: hola@danicallero.es

Más información: https://www.contributor-covenant.org/es/version/2/0/code_of_conduct/
